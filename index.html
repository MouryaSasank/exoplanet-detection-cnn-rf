The Mission
Why Exoplanet Detection Matters
One of astronomy's hardest classification problems — finding hidden worlds in starlight
🌍

What Are Exoplanets?

Planets orbiting stars beyond our Solar System. Over 5,000 confirmed, but billions more await discovery in Kepler's photometric data.

🔭

The Kepler Mission

NASA's telescope monitored 150,000+ stars, recording brightness every 30 minutes. Transit dips — tiny 0.01% drops — reveal hidden worlds.

⚖️

The Imbalance Challenge

Only 37 confirmed exoplanets vs 5,050 false positives. A naive model gets 99% accuracy by predicting "no planet" every time.

How It Works
End-to-End Pipeline
From raw light curves to exoplanet predictions
📡
Raw Kepler Light Curves

Data Ingestion

3,198 flux measurements per star from NASA's Kepler Space Telescope photometric observations.

🌊
Gaussian Denoising

Noise Reduction

Gaussian filter smooths stellar variability and instrument noise while preserving transit dip signatures.

🧠
1D CNN Feature Extraction

Deep Learning

3-layer CNN with 32→64→128 filters learns 64-dimensional transit feature embeddings automatically.

⚗️
SMOTE Balancing

Class Balancing

Synthetic Minority Over-sampling at 0.1 ratio addresses 37 vs 5,050 class imbalance in CNN feature space.

🌲
Random Forest Classifier

Ensemble Classification

200 trees with depth=8, balanced class weights, and isotonic probability calibration.

🪐
Exoplanet Prediction

Final Output

Optimized threshold maximizes F1-score while maintaining precision ≥ 0.4 for reliable detection.

Model Architecture
CNN + Random Forest Hybrid
CNN extracts 64-dimensional feature embeddings, Random Forest classifies them
1D CNN — Feature Extractor
Conv1D(32, kernel=7) + ReLU + L2 // broad transit dips
↓ BatchNorm → MaxPool(2)
Conv1D(64, kernel=5) + ReLU + L2 // refined features
↓ BatchNorm → MaxPool(2)
Conv1D(128, kernel=3) + ReLU + L2 // fine patterns
↓ BatchNorm → MaxPool(2)
GlobalAveragePooling1D
↓
Dense(64, ReLU) // ★ feature layer
↓ Dropout(0.4)
Dense(1, Sigmoid) // focal loss
Hybrid RF — Classifier
64-dim CNN Features // from feature_layer
↓ SMOTE (0.1 ratio)
StandardScaler // zero-mean, unit-var
↓
RandomForest(200 trees, depth=8) // balanced weights
↓
CalibratedClassifierCV // isotonic calibration
↓
Threshold Optimization // F1-max, prec≥0.4
↓
Final Prediction // exoplanet or not
